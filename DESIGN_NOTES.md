# ABA · Libi 발표덱 — 디자인 노트 (v2, 새로 제작)

frontend-design 스킬 + 웹 리서치(슬라이드 디자인 원칙) 기반.

## 적용한 원칙 (리서치 대조용 체크리스트)
- [ ] **그리드 정렬**: 모든 제목/이미지/카드가 동일 좌측 기준선·열 그리드에 정렬 (프로/아마추어 차이의 핵심)
- [ ] **여백 15~20%**: 슬라이드 가장자리·요소 사이 숨 쉴 공간. 채우지 말 것
- [ ] **시각 위계**: kicker → 제목(가장 큼) → lead → 본문 순. 한 슬라이드 한 메시지
- [ ] **60-30-10**: 종이/잉크(중립) 60 · 네이비(브랜드) 30 · 골드(악센트) 10. 티얼/레드 극소량
- [ ] **폰트 2(+1)**: Pretendard(본문·제목) + Fraunces(라틴 악센트) + Plex Mono(라벨). 화면=산세리프 우선
- [ ] **다이어그램**: 레이어로, 핵심 컴포넌트·관계만, 흐름 방향 표시, 과부하 금지
